# Writeup: From Syntax to Circuits

LaTeX source for the project paper, *"From Syntax to Circuits: Can a Quantum
Circuit Read a Sentence?"*, formatted with the official **ICML 2026** style.

## Contents

- `main.tex` — the paper.
- `references.bib` — bibliography.
- `figures/` — figures (grammar diagram, two-qubit circuit, training loss),
  generated by the project pipeline (`python -m qnlp_tutorial.demo`).
- `icml2026.sty`, `icml2026.bst`, `fancyhdr.sty` — the official ICML 2026
  template files
  (from <https://media.icml.cc/Conferences/ICML2026/Styles/icml2026.zip>).

## Building

Requires a LaTeX distribution with `pdflatex` and `bibtex` (e.g. MacTeX / TeX
Live). Then:

```bash
make          # produces main.pdf
make clean    # remove build artifacts
```

The document is built in **preprint** (non-blind) mode, so author names are
shown. For a blind submission version, change `\usepackage[preprint]{icml2026}`
to `\usepackage{icml2026}` in `main.tex`; for camera-ready, use
`\usepackage[accepted]{icml2026}`.

## Refreshing the figures

The figures are copies of `../poster_assets/*.png`. To regenerate them from the
model, run the demo from the repository root:

```bash
python -m qnlp_tutorial.demo
```

then copy `poster_assets/{grammar_diagram,toy_circuit,loss_curve}.png` into
`figures/`.
